## 개요

-   공공 데이터를 활용한 아파트 매물 검색 웹 애플리케이션
-   JWT 기반의 사용자 세션 관리 기능
-   Open API를 사용한 사용자 편의성 증대

## 개발환경

-   ### Server

    -   SpringBoot REST Server
    -   Spring Tool Suite

-   ### Database

    -   MySQL
    -   MyBatis

-   ### Client
    -   Vue.js
    -   HTML 5
    -   Bootstrap 4
    -   VSCODE
